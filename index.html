NUNITO

INTRODUCTION

Nunito was created by an English designer named Vernon Adams. Vernon has worked on Pacifico, Oxygen, Rokkitt, and a couple more fonts in his lifetime. He created this font in 2011 to be used as a display font. Nunito is a sans serif font and has a more rounder bubbly feel to it. This font has a range of weights from extra light to black. There is another version of Nunito called Nunito Sans which was created by Jacques Le Baliey. This font is a non rounded version of Nunito.
